# Parallel parking
Проект для курса "Неклассические логики" - параллельная парковка

## Описание
Спроектировать и реализовать экспертную систему, принимающую решения по параллельной парковке автомобиля с произвольным заполнением парковочных мест.
### Входные данные: 
- карта заполнения пространства
- радиус разворота (относительно габаритов автомобиля)
- координаты начального положения автомобиля
- ориентация объекта "по сторонам света" на карте заполнения
- дальность сенсора препятствий
- максимальная скорость объекта
- направление движения (вперед или назад)

Система управляет скоростью и поворотами объекта
